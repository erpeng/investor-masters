# 看懂边界 vs 品味判断

[段永平](/investor-masters/investors/duan-yongping/) 的起点是边界感。看不懂，就不碰；真正重要的不是知道得更多，而是知道哪些东西不该碰。[纳瓦尔·拉维坎特](/investor-masters/investors/naval-ravikant/) 的起点则更像一个持续训练中的内部罗盘: 先进入竞技场，在经验里反复试错，再把这些具体经验压缩成判断力，最后变成“说不清但知道对不对”的品味。

这并不是保守与激进的简单对立。更深的分歧在于，他们对“好判断”从哪里来，有不同回答。段永平更相信收缩决策空间，让判断只发生在真正进入能力圈的地方；纳瓦尔则更相信，通过长期做事、反思、阅读和识人，可以逐步把判断训练成一种更宽的系统能力。

如果你常常在“应该先收窄边界”还是“应该持续训练自己的判断力”之间摇摆，这一组特别值得反复读。它逼你回答一个更底层的问题: 你的优势，到底来自少犯错，还是来自把自己训练成一个更会做难判断的人。
