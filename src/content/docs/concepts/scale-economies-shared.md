# 共享规模经济

## 定义

共享规模经济指企业不把规模优势全部变成短期利润，而是持续把效率提升返还给客户，借此增强客户忠诚度、扩大市场份额，最终形成更深的护城河。

## 谁最深刻地讨论了这个概念

[尼克·斯利普](/investor-master/investors/nick-sleep/) 与 [凯斯·扎卡里亚](/investor-master/investors/qais-zakaria/) 讨论得最系统。他们把 [开市客](/investor-master/companies/costco/) 和 [亚马逊](/investor-master/companies/amazon/) 看成这一机制的经典案例。[莫尼什·帕伯莱](/investor-master/investors/monish-pabrai/) 在谈开市客时也给出了非常接近的表达。
